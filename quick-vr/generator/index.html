Quick VR Generator

Quick VR is a web Virtual Reality Scene generator using Google Spreadsheets for configuration

Usage

  1. Create a Google Spreadsheet. You can use a simple template
  2. Publish it for the web
  3. Paste the url you get here and hit
  4. you can:
    • Share your scene link: ...
    • place an iframe in your website

Examples

Most examples are taken from the webvr-starter-kit repo.

  • Simple scene
  • Panorama
  • 3D video sphere

Docs (WIP)

Intro

This tool is just a simple wrapper for webvr-starter-kit. You can read the webvr-starter-kit for more examples and usage.

Objects

All properties from webvr objects are allowed.
